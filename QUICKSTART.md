# Quick Start Guide

## Get the App Running in 3 Steps

### 1. Install Dependencies
```bash
cd /Users/mekari/Workspace/belajar/media_player/MediaPlayer
npm install
```

### 2. Start Metro Bundler
```bash
npm start
```

### 3. Run on Android (in a new terminal)
```bash
npm run android
```

That's it! The app should now be running on your Android device/emulator.

## Testing Android Auto

### Fastest Way: Phone + Android Auto App

1. Install "Android Auto" from Play Store
2. Open Android Auto > tap menu 10 times to enable developer mode
3. Settings > Developer Settings > Enable "Unknown sources"
4. Open Android Auto while your app is running
5. Your app will appear in the media section!

## What You Can Do

- **Home Tab**: Browse and play songs
- **Library Tab**: Access playlists and albums
- **Mini Player**: Control playback from any screen
- **Full Player**: Tap mini player for full controls
- **Android Auto**: Use your app in the car or with DHU
